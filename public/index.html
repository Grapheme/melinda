What is it?

Melinda — is a Chrome-compatible experimental web application created by the members of Grapheme team. Melinda lets you create Scopes — interactive kaleidoscopic visualizations that are moving along with the music.

How it works?

  1. Drag-n-Drop your favourite image to change the pattern of a kaleidoscope.
  2. Do the same trick to change the music.
  3. Share the link to this Scope with your friends!
